## Installation

NSJTextField is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following lines to your Podfile:

```ruby
source 'https://github.com/NSJoker/NSJTextFieldSpecs.git'
pod 'NSJTextField'
```
